## 1. 什么是绿色免安装版

本目录是第二课堂管理系统的 **绿色免安装版本**（也称"便携版" / Portable Edition），由 `electron-builder` 构建生成。解压后无需安装，双击主程序即可直接运行。

**版本**: `v1.0.0`
**构建日期**: 2026-06
**适用平台**: Windows 10 / Windows 11（64 位）

## 2. 与安装版的区别

| 对比项 | 绿色免安装版（本目录） | 安装版（Setup.exe） |
|--------|------------------------|---------------------|
| 是否需要安装 | 否，解压即用 | 是，运行安装向导 |
| 是否创建桌面快捷方式 | 否，需手动创建 | 是，自动创建 |
| 是否写入注册表 | 否 | 是 |
| 是否创建开始菜单项 | 否 | 是 |
| 数据存储位置 | 与程序同目录的 Electron 用户数据 | 系统 `%APPDATA%` 目录 |
| 能否放在 U 盘运行 | 是 | 否 |
| 卸载方式 | 直接删除文件夹 | 通过控制面板卸载 |
| 文件体积 | 约 250 MB | 约 120 MB（压缩包） |

**绿色版适用场景**：

- 需要在多台电脑间用 U 盘携带使用
- 没有系统管理员权限，无法运行安装程序
- 临时使用，不希望留下系统痕迹
- 部署到公共机房或多媒体教室

## 3. 如何使用

### 3.1 首次启动

1. 确保本目录完整解压（所有文件和 `locales`、`resources` 子目录齐全）
2. 双击 **`第二课堂管理系统.exe`** 启动程序
3. 首次启动会自动初始化默认账户，稍等片刻即可看到登录界面

### 3.2 创建桌面快捷方式（可选）

- **Windows 10 / 11**：右键 `第二课堂管理系统.exe` → 发送到 → 桌面快捷方式
- 之后直接从桌面双击快捷方式启动即可

### 3.3 关闭程序

- 直接点击窗口右上角 × 关闭
- 或在应用内点击右上角"退出"按钮
- 如果启用了"关闭时自动备份"功能，关闭前会自动下载一份 JSON 备份文件

## 8. 数据存储位置

本程序为纯前端应用，**所有数据存储在本地，不上传任何服务器**。

### 绿色版的数据存储路径

绿色版使用 Electron 默认的 `userData` 目录，位于：

```
%APPDATA%\第二课堂管理系统\
```

完整路径通常为：

```
C:\Users\<你的用户名>\AppData\Roaming\第二课堂管理系统\
```

### 存储内容

在该目录下，浏览器 `localStorage` 数据持久化为 LevelDB 格式（Chromium 引擎自动管理）。包含以下业务数据：

| 数据项 | 说明 |
|--------|------|
| 用户账户（密码哈希） | 所有可登录用户 |
| 登录状态 | 当前会话（sessionStorage） |
| 处理结果数据 | 最近一次 Excel 数据处理结果 |
| 处理历史 | 最多 20 条历史记录 |
| 公告内容 | 系统通知公告 |
| 操作日志 | 最多 500 条操作记录 |
| 达标规则配置 | 各类别阈值设定 |
| 实施办法内容 | 制度文件正文 |
| 主题偏好 | 浅色 / 深色 / 跟随系统 |

### 清除数据

如需彻底清除所有数据：
1. 在程序设置中点击"清除数据"按钮
2. 或手动删除 `%APPDATA%\第二课堂管理系统\` 目录
3. 下次启动程序将恢复为初始状态

## 9. 备份与迁移

### 日常备份

在程序的**设置 → 备份恢复**中：

1. 点击 **"导出备份"** → 下载完整 JSON 备份文件（包含账号、数据、日志、配置）
2. 点击 **"导出配置包"** → 下载仅配置的 JSON 文件（账号 + 公告 + 实施办法，不含个人数据）

### 迁移到另一台电脑

1. 在旧电脑上，进入程序**设置 → 备份恢复 → 导出备份**，保存 JSON 文件
2. 将绿色版文件夹和 JSON 备份文件复制到新电脑
3. 在新电脑上启动程序，进入**设置 → 备份恢复 → 导入备份**，选择 JSON 文件
4. 导入成功后刷新页面即可看到全部数据

### 关闭时自动备份

如果启用了"关闭时自动备份"功能（默认开启），每次关闭程序时会自动下载一份完整备份 JSON 文件，建议定期收集这些备份文件并存放到安全位置。

## 10. 升级方法

### 从旧绿色版升级

1. 在旧版本中**导出备份**（设置 → 备份恢复 → 导出备份）
2. 关闭旧版本程序
3. 删除旧的绿色版文件夹
4. 解压新版本绿色版到同一位置
5. 启动新版本，在设置中**导入备份**
6. 验证数据完整性后即可正常使用

### 从安装版迁移到绿色版

1. 在安装版中**导出备份**
2. 卸载安装版（或保留）
3. 解压绿色版
4. 启动绿色版，**导入备份**

> **注意**：绿色版和安装版的数据存储路径不同，需要通过手动导出/导入 JSON 文件来迁移数据。

## 11. 常见问题

### Q: 为什么有两个版本（安装包 + 绿色版）？

> 安装包提供完整的系统集成体验（桌面快捷方式、开始菜单、卸载入口），适合固定电脑长期使用。绿色版无需安装、不写注册表，适合 U 盘携带或临时使用。

### Q: 绿色版运行时杀毒软件报警怎么办？

> 由于 Electron 框架使用 V8 引擎和 Chromium 内核，某些杀毒软件可能误报。本程序使用开源框架构建，不含恶意代码。如遇拦截，请在杀毒软件中将程序目录添加到信任区。

### Q: 能否在 U 盘中直接运行？

> 可以。将整个 `win-unpacked` 文件夹复制到 U 盘，在其他 Windows 10/11 电脑上插入 U 盘后双击 exe 即可运行。注意 U 盘读写速度会影响启动速度，建议使用 USB 3.0。

### Q: 数据会保存到 U 盘还是电脑？

> 由于 Electron 的 `userData` 默认指向系统 `%APPDATA%` 目录，数据会保存在运行该程序的电脑上，而非 U 盘。如果在多台电脑上使用，每台电脑的数据是独立的。如需同步，请通过备份/恢复功能手动迁移。

### Q: locales 文件夹里的语言包能不能删？

> 可以只保留 `zh-CN.pak`（简体中文）和 `en-US.pak`（英语回退），删除其余 .pak 文件可节省约 20 MB 空间。但建议保留所有文件以确保完整功能。

### Q: 程序占内存大怎么办？

> Electron 本质是内嵌 Chromium 浏览器，启动约占用 150-200 MB 内存，属于正常范围。处理大量 Excel 数据时内存占用会上升。如果内存紧张，建议关闭其他不必要的程序。

### Q: locales 里的文件是什么？全部都要吗？

> `locales/` 是 Chromium 的多语言界面包，仅影响 Electron 内置对话框（如文件选择器）的界面语言。程序自身的界面语言为中文，由前端代码控制。技术上可以只保留 `zh-CN.pak`，但建议不要修改。
